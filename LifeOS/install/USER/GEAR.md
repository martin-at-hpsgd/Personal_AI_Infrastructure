# My Gear

> **Single source of truth for everything you own and use — physical kit and tech stack.** This is your canonical inventory. Any skill that touches gear (coffee, audio, network, smart home, etc.) READS this file and WRITES its updates back here. Never keep a private gear copy inside a skill.
>
> **Scope: gear only.** Physical possessions and the tech stack you build on. NOT security posture or system internals.
>
> This is a **bootstrap template** — replace the example rows with your own gear, or run `/interview`. Sections marked **TODO** are stubs waiting for you. Add gear under the right heading; date significant changes. The Pulse **Assets** tab renders whatever is in this file.
