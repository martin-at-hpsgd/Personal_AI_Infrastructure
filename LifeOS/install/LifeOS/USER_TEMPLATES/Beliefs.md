# Beliefs

What you take to be true after long attention. Each section captures a position worth defending.

## On knowledge

Knowledge is a *hard-to-vary explanation* — a description of reality where every detail plays a functional role, so contrary evidence has nowhere to flee.

## On consciousness

Consciousness is what it feels like to be a sufficiently complex information-processing system observing itself.

## On meaning

Meaning is not given. It is constructed, and the construction is the point.
