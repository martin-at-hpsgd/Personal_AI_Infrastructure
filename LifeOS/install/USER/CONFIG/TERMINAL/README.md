# TERMINAL — Terminal Preferences

This directory holds settings that customize how LifeOS integrates with
your terminal — tab naming, environment persistence, color preferences.
Most users never touch it; the defaults work for Kitty, Ghostty, iTerm2,
and Terminal.app.

## When to edit

- You use a terminal emulator that needs custom escape sequences.
- You want different tab-rename behavior across multiple LifeOS sessions.
- You're integrating a launcher (Alfred, Raycast) that needs LifeOS metadata.

## Privacy

Nothing in this directory ships in a public LifeOS release. Edit freely;
your customizations stay local.
